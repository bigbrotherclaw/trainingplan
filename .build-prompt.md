You are rebuilding this training plan web app from scratch. The existing index.html has the full original app - read it carefully to understand the training program structure, workout types, exercise data, and periodization scheme. Extract ALL data.

## GOAL
Build a professional-grade, mobile-first training plan PWA that is a MAJOR upgrade over the existing single-file app.

## TECH STACK
- React 18 + Vite
- Tailwind CSS 3
- Framer Motion (smooth page transitions, animations)
- Recharts (for ALL charts and data visualizations)
- @dnd-kit/core + @dnd-kit/sortable (workout day swapping)
- Lucide React (icons)
- date-fns (date handling)
- localStorage for persistence with JSON export/import

## WHAT TO KEEP FROM THE ORIGINAL
- The exact same training program structure (TB Operator, 6-week periodization)
- OPERATOR_LOADING cycle (weeks 1-6 with sets/reps/percentages)
- All 3 main lifts (Bench, Squat, Weighted Pull-up)
- All accessories (A/B/C groups) with every exercise
- All HIC presets with their categories and descriptions
- All cardio presets (swim, bike, run) with week-based progression
- Endurance presets (bike and run)
- The weekly template (Sun=rest, Mon=STR A, Tue=Swim/Bike+HIC, Wed=STR B, Thu=Run+HIC, Fri=STR C, Sat=Long Tri)
- Settings for 1RM values, lactate thresholds, FTP, block/week
